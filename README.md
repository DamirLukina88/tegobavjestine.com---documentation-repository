# TEGOBA VJESTINE - Documentation & Assets

Welcome to the central documentation and design repository for **TEGOBA VJESTINE**.

This repository is dedicated entirely to project planning, system architecture documentation, and visual assets. **There is no source code in this repository.**

## 📚 Project Wiki

Detailed documentation, game mechanics, technical guides, and lore are maintained in the GitHub Wiki.

👉 **[Go to the TEGOBA VJESTINE Wiki](https://github.com/DamirLukina88/tegobavjestine.com---documentation-repository/wiki)**

## 🎨 Visual Assets

This repository stores all visual assets organized into the `visuals/` directory:

*   `visuals/mockups/` - UI/UX designs, wireframes, and interface prototypes.
*   `visuals/diagrams/` - System architecture, database schemas, and flowchart diagrams.
*   `visuals/assets/` - Logos, branding, character art, and other graphical assets.
